# Communications LTD

## Installation 

Clone the repository <br/>
`$` `git clone https://github.com/GuyGooL5/communication-ltd.git`

Install Modules <br/>
`$` `npm install` <br/>
`$` `cd client`<br/>
`$` `npm install`<br/>
`$` `cd ..`<br/>

## Backend dev server

To start the backend dev server run <br/>
`$` `npm start`

## Frontend dev server

To start the frontend dev server run <br/>
`$` `cd client` <br/>
`$` `npm install` <br/>
