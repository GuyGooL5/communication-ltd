# Communications LTD

## Installation 

Clone the repository 
`$` `git clone https://github.com/GuyGooL5/communication-ltd.git`

Install Modules
`$` `npm install`
`$` `cd client`
`$` `npm install`
`$` `cd ..`

## Backend dev server

To start the backend dev server run
`$` `npm start`

## Frontend dev server

To start the frontend dev server run
`$` `cd client`
`$` `npm install`
